🔒 Segurança dos seus dados

Esta versão não zera seus cadastros automaticamente. Os dados ficam salvos no navegador usando localStorage. Antes de trocar de versão, use Exportar Backup. Depois abra a nova versão e use Importar Backup.

Regra prática: sempre baixe um backup antes de substituir o arquivo HTML por uma versão nova.
